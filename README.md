# my-xv6

> 此项目为南京航空航天大学操作系统课程实验内容

---

每个实验是一个单独的分支：

* proj0: 添加用户命令、添加内核输出语句、添加系统调用
* proj1: 带参数的系统调用、子进程优先的fork
* proj2: 线程支持、优先级调度
* proj3: 信号量、消息传递
* proj4: 写时复制的fork、复用内核页表
* proj5: 文件系统校验

使用`git checkout proj<n>`来查看每个实验的代码和报告。
